## 📱 ANLEITUNG iPhone / iPad

### Schritt 1: Safari öffnen
**WICHTIG:** Nicht Chrome! Nicht Firefox! **NUR Safari** funktioniert auf iPhone.

### Schritt 2: URL eingeben
Geben Sie ein:
```
specialty-key-singer-numeric.trycloudflare.com
```
oder lassen Sie den Computer einen QR-Code zeigen (Dashboard ganz unten scrollen).

### Schritt 3: 30 Sekunden warten + scrollen
Die App muss komplett geladen sein. Scrollen Sie kurz auf der Startseite.

### Schritt 4: Teilen-Symbol antippen
Unten in der Mitte des Safari-Bildschirms ist ein Symbol:

```
┌─────────────────────┐
│                     │
│    [App-Inhalt]     │
│                     │
├─────────────────────┤
│  ◀  ▶   📤   📑  📚  │  ← TIPPEN auf 📤
└─────────────────────┘
```

Es sieht aus wie ein Kasten mit Pfeil nach oben **⬆**.

### Schritt 5: "Zum Home-Bildschirm" wählen
Im Menü das jetzt erscheint, **NACH UNTEN scrollen** bis Sie sehen:
```
🔼 Zum Home-Bildschirm hinzufügen
```
Antippen.

### Schritt 6: Bestätigen
Oben rechts auf **"Hinzufügen"** tippen.

✅ **FERTIG!** Auf dem Home-Bildschirm erscheint das App-Icon. Antippen → App startet im Vollbild ohne Browser-Leiste, wie eine echte App.

## ❓ Wenn das App-Logo nicht erscheint

### Problem: "Ich sehe das Icon nicht im Menü"

**iPhone-Lösung:**
1. App-Seite mind. **30 Sek lang** offen lassen
2. Zwischen verschiedenen Bereichen klicken (z.B. „Familie" → „Gesundheit")
3. Erst DANN auf Teilen-Symbol → "Zum Home-Bildschirm"

**Android-Lösung:**
1. Stellen Sie sicher, dass Sie **Chrome** benutzen (nicht Samsung Internet)
2. Chrome **ist aktuell**? (Play Store → Chrome → "Aktualisieren")
3. Browser-Einstellungen → "Apps installieren" muss erlaubt sein

### Problem: "App-Icon ist weiß / leer"
Das passiert manchmal beim ersten Mal. Lösung:
1. App-Icon vom Home-Bildschirm löschen
2. Browser-Cache leeren (Einstellungen → Browser → Cache löschen)
3. URL neu öffnen, **2 Minuten warten**
4. Dann erneut "Zum Home-Bildschirm" → diesmal ist das Logo da

## ⚠️ Hinweis zur Cloudflare-URL

Diese URL `specialty-key-singer-numeric.trycloudflare.com` ist **temporär**. Solange der Computer an ist, läuft sie. Sobald der Computer aus ist oder der Tunnel neu startet, ist die URL kaputt.

**Damit die App-URL für IMMER funktioniert** (auch wenn der Computer aus ist):
- ZIP-Paket auf Ihrem Desktop: `FamilienApp-Deploy.zip`
- Auf https://app.netlify.com/drop ziehen → permanente URL für immer
- Diese permanente URL dann auf dem Handy installieren

Aber für den Anfang reicht die Cloudflare-URL!
