# FamilienApp

Eine Progressive Web App (PWA) für Familien in Deutschland — mit Zuschüssen &
Anträgen, Sparen & Geldanlagen, Umgebungssuche, Rezepten, Gesundheit, Kalender,
Erziehungshilfe, Senioren-Bereich, Foto-Album, Kontakten und vielem mehr.

**App öffnen:** https://marcel-fe.github.io/familien-app/

Die App lässt sich ohne App-Store direkt auf dem Handy installieren:
- **iPhone:** Link in Safari öffnen → Teilen → „Zum Home-Bildschirm".
- **Android:** Link in Chrome öffnen → Menü → „App installieren".
